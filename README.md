# 365 Seguridad
**Este trabajo se creó a base de una colaboración entre mi persona y un chico emprendedor en el mundo de la seguridad. Donde realiza instalación de cámaras de seguridad, sercos electrónico, alarmas comunitarias, y planes de internet más TV cable.**

## El Diseño
Los trazos de cada sección los hice en un 📔 cuaderno para después pasarlo a digital y me gusto mucho usar **Lunacy** una alternativa -free- a *Figma*.

Prefiero Lunacy porque me da la facilidad de trabajar sin conexión a internet mas la creación de archivos *locales*, tiene todo de figma lo que hace perfecto para *MI* para pasar los trazos a digital.

Para ello hice:
1. Arme la paleta de colores.
2. Descargue dos fuentes de *[Google Fonts](https://fonts.google.com/)*.
3. Diseñe el logo con *[Affinity](https://www.affinity.studio/)*.
4. La imágenes son propias de *[Unsplah](https://unsplash.com/)*.

## El Desarrollo

Para el desarrollo utilicé:
1. Cree el proyecto con [Bun](https://bun.com/).
2. Elegí el meta framework preferido 🚀[Astro](https://astro.build/).
3. Optimice las imágenes en [Squoosh](https://squoosh.app/).
4. Esta adaptado para móvil 📱, laptop 💻, desktop y ➡️ otras resoluciones mas grandes 🖼. 
5. Y de momento esta desplegado en 🔼[Vercel](https://vercel.com/).
